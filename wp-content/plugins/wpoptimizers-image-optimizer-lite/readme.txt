== Description ==

WPOptimizers - Image Optimizer Lite is a lightweight and powerful WordPress image optimization plugin that helps you improve your website speed by compressing and optimizing images directly inside your WordPress dashboard. With its simple interface and automatic optimization, you can boost your site’s performance, SEO rankings, and Core Web Vitals in just a few clicks. 

Built for simplicity, speed, and compatibility, this plugin ensures your images load faster without compromising quality. Perfect for bloggers, businesses, and developers who care about website performance and user experience.  

= Features =
* One-click image optimization – optimize all your images instantly
* Supports JPEG, PNG, and GIF images
* Lightweight and fast – no bloat, works with any theme or plugin
* Optimizes new uploads automatically
* Reduces file size without visible quality loss
* Improves SEO, Core Web Vitals, and page speed scores
* Media Library badge to show which images are optimized or not (NEW)
* Bulk optimization fixes with proper status saving (NEW)

= Why choose WPOptimizers Lite? =
Unlike bulky image optimization plugins that require complex configurations or external APIs, WPOptimizers Lite is designed for users who want a **simple, free, and effective WordPress image optimization solution**.  

Whether you’re a blogger, business owner, or developer, this plugin offers:  
- A clean and beginner-friendly interface  
- Smart image compression for faster loading times  
- Improved site performance without extra server load  
- Better search engine rankings with optimized images  

If you need a **fast, lightweight, and beginner-friendly WordPress image optimizer**, WPOptimizers Lite is the perfect choice to make your website load faster and deliver a smoother user experience.

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/wpoptimizers-image-optimizer-lite` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress.
3. Go to **WPOptimizers** from the WordPress admin menu.
4. Start optimizing your images with one click!

== Frequently Asked Questions ==

= Will this plugin slow down my site? =
No. WPOptimizers Lite is lightweight and optimized for performance. It runs efficiently in the background without adding extra load to your server.

= Do I need an API key or account to use this plugin? =
No. WPOptimizers Lite works out of the box — no API key, signup, or third-party service required.

= Is there a limit on how many images I can optimize? =
No, you can optimize unlimited images without restrictions in the Lite version.

= Does it work with WooCommerce or page builders? =
Yes! The plugin works seamlessly with WooCommerce, Elementor, Divi, Gutenberg, and all major page builders and themes.

= Will my images lose quality after optimization? =
No. The plugin uses smart compression technology to reduce file size while preserving image quality. Your images will look visually the same.

= Can I optimize existing images in my media library? =
Yes, you can bulk optimize your existing images directly from your WordPress dashboard with one click.

= Is this plugin free? =
Yes! WPOptimizers Lite is completely free. A Pro version with advanced features may be available in the future.
