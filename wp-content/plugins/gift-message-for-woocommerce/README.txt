## Gift Message

The Gift Message for WooCommerce plugin lets you add a gift message to your wooCommerce checkout page.
Your customers can write a gift message for their orders, and you can easily copy the gift message in the admin panel.

Your customers will view the gift messages on order thank you page and emails. The store owner will view it on the admin panel.

This plugin is handy to any wooCommerce store that wants to give customers a better user experience in adding gift messages to the order on the checkout page.

## Premium Features

[Gift Message for WooCommerce Premium](https://powerfulwp.com/gift-message-for-woocommerce-premium/) includes the following additional features:

* Add occasions to your wooCommerce checkout page.
* 25 occasions to choose from on the checkout page.
* Occasions manager - you can update, delete and add many occasions as you want.
* Gift message suggestions for your customers on the wooCommerce checkout page.
* 72 New Gift Message Suggestions sorted by occasion.
* Gift message suggestions manager - you can update, delete and add many gift message suggestions as you want.
* Admin can copy the gift message with one click.
* Admin can select where to show the gift message on the checkout page.
* Gif message column on the admin panel orders page, copied with one click.
* Gif message cards - Let your customers buy a gif message card at the checkout page.

== Installation ==

1. Make a backup of your website and database
1. Download the plugin
1. Upload the plugin to the wp-content/plugins directory,
1. Go to “plugins” in your WordPress admin, then click activate.
1. You will now see the Gift Message option on your left navigation bar.
1. On the general setting page, you can select if the Gift message field is required and set the maximum length of the gift message.
